# Phase 8: Order Execution (Stocks, Crypto, Options)


## Overview

Full trading capability via Alpaca: equities, crypto (24/7), and options. Order entry UI with type selection, confirmation flow, and position management.

**Deploy outcome**: Users place/cancel orders for stocks, crypto, options. View order status. Close positions.


## Context

Phase 7 established templates. Phase 8 enables acting on trade recommendations.

**Source code (mahoraga)**:
- `src/providers/alpaca/trading.ts`: order CRUD, positions, account
- `src/providers/alpaca/options.ts`: options chain, contracts, snapshots
- `src/mcp/agent.ts`: orders-preview, orders-submit flow

---


## Goals & Non-Goals

**Goals**:
- Port Alpaca trading client (per-user credentials)
- Stock orders (market, limit, stop, stop-limit)
- Crypto orders (24/7, BTC/ETH/SOL)
- Options orders (calls/puts, chain lookup)
- Delta targeting for options
- Order entry UI with type picker
- Order confirmation before submit
- Position close (partial/full)
- Order cancellation

**Non-Goals**:
- Complex order types (OCO, bracket) - Phase 10
- Trailing stops (defer)
- Options spreads (defer)
- Pre/post market for stocks (defer)

---


## Features

### 8.1 Order Placement Service

Create orders via Alpaca using user's credentials.

### 8.2 Order Types

Market, limit, stop, stop-limit for stocks. Market/limit for crypto and options.

### 8.3 Stock Orders

Standard equity orders with extended hours support (future).

### 8.4 Crypto Orders

BTC/USD, ETH/USD, SOL/USD. 24/7 trading. GTC time-in-force.

### 8.5 Options Orders

Single-leg options orders. Contract lookup via chain.

### 8.6 Options Chain Endpoint

Get expirations, strikes, with greeks.

### 8.7 Delta Targeting

Filter options by delta range (0.3-0.7 for practical strikes).

### 8.8 Order Entry UI

Symbol input, side, qty/notional, type, price fields.

### 8.9 Options Selector UI

Expiration picker, strike picker, delta filter.

### 8.10 Order Confirmation

Review order details before submit.

### 8.11 Position Close UI

Quick close button per position (full or partial).
