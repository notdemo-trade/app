# Phase 5: Technical Analysis


## Overview

Calculate technical indicators from Alpaca market data, detect trading signals, display on charts.

**Deploy outcome**: Users view TA indicators (SMA, EMA, RSI, MACD, Bollinger, ATR) for watchlist symbols with signal detection.


## Context

Phase 4 established signal gathering from social sources. Phase 5 adds market-based technical signals.

**Source code (mahoraga)**:
- `src/providers/technicals.ts`: indicator calculations, signal detection
- `src/providers/alpaca/market-data.ts`: bars/quotes fetching

---


## Goals & Non-Goals

**Goals**:
- Port TA calculations from mahoraga
- Fetch market data (bars) from Alpaca per-user
- API endpoint for indicators + signals
- Price chart with indicator overlays
- Signal markers on chart

**Non-Goals**:
- Real-time streaming (WebSocket)
- Custom indicator periods (fixed defaults for MVP)
- Alert triggers (Phase 16notifications)

---


## Features

### 5.1 TA Calculation Service

Port indicators: SMA (20/50/200), EMA (12/26), RSI (14), MACD (12/26/9), Bollinger (20,2), ATR (14).

### 5.2 Market Data Fetcher

Get historical bars from Alpaca using user's credentials.

### 5.3 TA Endpoint

`GET /api/analysis/:symbol` - returns indicators + detected signals.

### 5.4 Price Chart Component

Candlestick chart using lightweight-charts or similar.

### 5.5 Indicator Overlay

Show SMA/EMA/Bollinger on price chart, RSI/MACD in separate panels.

### 5.6 Signal Markers

Visual markers for detected signals (RSI oversold, MACD cross, etc.).

## Type Definitions

```ts
// packages/data-ops/src/providers/technicals/types.ts

export interface Bar {
  t: string    // ISO timestamp
  o: number    // Open
  h: number    // High
  l: number    // Low
  c: number    // Close
  v: number    // Volume
  n: number    // Trade count
  vw: number   // VWAP
}

export interface TechnicalIndicators {
  symbol: string
  timestamp: string
  price: number
  sma_20: number | null
  sma_50: number | null
  sma_200: number | null
  ema_12: number | null
  ema_26: number | null
  rsi_14: number | null
  macd: {
    macd: number
    signal: number
    histogram: number
  } | null
  bollinger: {
    upper: number
    middle: number
    lower: number
    width: number
  } | null
  atr_14: number | null
  volume_sma_20: number | null
  relative_volume: number | null
}

export type SignalDirection = "bullish" | "bearish" | "neutral"

export interface TechnicalSignal {
  type: string
  direction: SignalDirection
  strength: number // 0-1
  description: string
}

export interface AnalysisResult {
  symbol: string
  timeframe: string
  indicators: TechnicalIndicators
  signals: TechnicalSignal[]
  bars: Bar[]
}

export type Timeframe = "1Min" | "5Min" | "15Min" | "1Hour" | "1Day"
```
