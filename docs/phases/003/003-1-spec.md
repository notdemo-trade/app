# Phase 3: Account & Portfolio View


## Overview

Display Alpaca account info, positions, and orders through the platform using per-user credentials.

**Deploy outcome**: Users see account equity/buying power, positions with P&L, recent orders.


## Context

Phase 2 stores encrypted Alpaca credentials. Phase 3 uses those credentials to fetch and display portfolio data.

**Source code (mahoraga)**:
- `src/providers/alpaca/client.ts`: base HTTP client
- `src/providers/alpaca/trading.ts`: account, positions, orders methods
- `src/providers/types.ts`: Account, Position, Order interfaces

---


## Goals & Non-Goals

**Goals**:
- Port Alpaca client from mahoraga
- API endpoints for account/positions/orders
- Dashboard showing equity, positions, orders
- Handle error cases (no creds, invalid creds, API errors)

**Non-Goals**:
- Real-time streaming (future)
- Trading actions (Phase 8
- Historical analytics (beyond basic portfolio history)
- Multiple Alpaca accounts per user

---


## Features

### 3.1 Alpaca Client

Port from mahoraga with per-user credentials.

**Changes from mahoraga**:
- Remove env-based config (`ALPACA_API_KEY`)
- Accept credentials as constructor params
- Create client per-request using decrypted user credentials

### 3.2 Account Endpoint

`GET /api/account` - returns equity, cash, buying power, status

### 3.3 Positions Endpoint

`GET /api/positions` - returns current holdings with P&L

### 3.4 Orders Endpoint

`GET /api/orders` - returns recent orders with status

Query params:
- `status`: `open` | `closed` | `all` (default: `all`)
- `limit`: number (default: 50, max: 100)

### 3.5 Account Dashboard

Display: equity, cash, buying power, portfolio value, account status

**Features**:
- Auto-refresh every 30s (fixed interval)
- "Last updated" timestamp displayed
- KV cache for faster initial load (TTL: 30s)

### 3.6 Positions Table

Columns: symbol, qty, side, avg cost, current price, market value, unrealized P&L, P&L %

### 3.7 Orders Table

Columns: symbol, side, type, qty, filled qty, status, submitted at

### 3.8 Portfolio Chart (Optional)

Line chart of equity over time. Uses `/v2/account/portfolio/history`.
