# Phase 18: Production Launch


## Overview

Deploy notdemo.trade to production with security hardening, rate limiting, monitoring, and backups.

**Deploy outcome**: Live production site at notdemo.trade, ready for real users.


## Context

Phases 1-17 built and refined features on staging. Phase 18 launches to production.

---


## Goals & Non-Goals

**Goals**:
- Security audit (auth, encryption, validation)
- Per-user API rate limiting
- Production domain setup
- Environment secrets configuration
- Error tracking + monitoring
- Database backup strategy
- Production deployment

**Non-Goals**:
- Load testing (acceptable scale: <100 users)
- CDN setup (Cloudflare handles)
- Custom SSL (Cloudflare handles)
- Multi-region deployment

---


## Features

### 18.1 Security Audit

Review auth flows, credential encryption, input validation, XSS/CSRF protection.

### 18.2 Rate Limiting

Per-user API request limits using Cloudflare Workers.

### 18.3 Production Domain

Configure notdemo.trade DNS + Cloudflare.

### 18.4 Production Secrets

All env vars set in Cloudflare dashboard.

### 18.5 Monitoring

Sentry for errors, Cloudflare Analytics for traffic.

### 18.6 Backup Strategy

Neon automated backups + point-in-time recovery.

### 18.7 Deploy Prod

Final deployment via GitHub Actions.
