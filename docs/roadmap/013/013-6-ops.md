## Security Considerations

### XSS Prevention (P0 Blocker)

User-provided journal content (notes, lessons learned) must be sanitized before storage to prevent XSS attacks.

**Install dependency**:
```bash
pnpm add isomorphic-dompurify
```

**Sanitization utility**:
```typescript
// packages/data-ops/src/utils/sanitize.ts
import DOMPurify from 'isomorphic-dompurify'

export function sanitizeUserInput(input: string): string {
  return DOMPurify.sanitize(input, {
    ALLOWED_TAGS: [],
    ALLOWED_ATTR: []
  })
}
```

**Apply in handlers**:
```typescript
// apps/data-service/src/hono/handlers/journal.ts
import { sanitizeUserInput } from '@repo/data-ops/utils/sanitize'

export const createJournalHandler = async (c: Context) => {
  const body = await c.req.json()
  const validated = CreateJournalSchema.parse(body)

  const sanitized = {
    ...validated,
    notes: validated.notes ? sanitizeUserInput(validated.notes) : undefined
  }

  const journalId = await createJournalEntry(c.env.DB, sanitized)
  return c.json({ journalId })
}

export const updateJournalHandler = async (c: Context) => {
  const body = await c.req.json()
  const validated = UpdateJournalSchema.parse(body)

  const sanitized = {
    ...validated,
    notes: validated.notes ? sanitizeUserInput(validated.notes) : undefined,
    lessonsLearned: validated.lessonsLearned ? sanitizeUserInput(validated.lessonsLearned) : undefined
  }

  await updateJournalEntry(c.env.DB, userId, journalId, sanitized)
  return c.json({ success: true })
}

// Sanitize memory rule descriptions
export const createMemoryRuleHandler = async (c: Context) => {
  const body = await c.req.json()
  const validated = CreateMemoryRuleSchema.parse(body)

  const sanitized = {
    ...validated,
    description: sanitizeUserInput(validated.description)
  }

  const ruleId = await createMemoryRule(c.env.DB, userId, sanitized)
  return c.json({ ruleId })
}
```

**Sanitize before database insert**:

- Lessons learned text

### Resolution: Sharpe Ratio

**Formula** (trade-level, simplified):
```
returns[] = trades.map(t => (t.exitPrice - t.entryPrice) / t.entryPrice)
avg = mean(returns)
std = stdDev(returns)
sharpe = std === 0 ? 0 : (avg / std) * sqrt(252)
```

**Where**: `packages/data-ops/src/queries/stats.ts` as `calculateSharpe(trades: Trade[]): number`

**Simplifications**:
- Risk-free rate = 0 (negligible for short-term trading)
- Annualized with sqrt(252) (trading days/year)
- Returns 0 when std=0 (no variance) or <2 trades
- Trade-level returns, not daily portfolio returns (simpler, sufficient for leaderboard ranking)

**Null handling**: Phases 14/15 aggregation should treat null Sharpe as 0 (new users with <2 trades get neutral score, not excluded).
