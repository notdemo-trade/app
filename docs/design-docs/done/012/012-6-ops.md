## Decisions

1. **3 agents MVP, 7 full**: ship recommendations first, add execution later
2. **Orchestrator coordinates**: delegates to sub-agents, doesn't do work itself
3. **Pull model**: orchestrator calls sub-agents, sub-agents don't push
4. **Strategy in orchestrator**: templates stored in orchestrator SQLite, passed to LLM per-request
5. **Entitlements (MVP)**: simple enabled/disabled flags. TA + LLM enabled by default, signal agents disabled
6. **Recommendations over trades**: MVP logs recommendations to SQLite for UI display. Phase 8 converts to trade proposals
7. **No gatherSignals in MVP**: TA writes to shared PG. `collectAllSignals()` queries PG directly
8. **RPC**: `getAgentByName()` for all inter-agent + worker→agent calls
9. **Real-time**: `useAgent()` hook replaces React Query polling
